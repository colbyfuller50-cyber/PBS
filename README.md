# Palmetto Background Screening (PBS) Website

## Overview
Static HTML/CSS website for Palmetto Background Screening (PBS) with multi-page structure:
- Home (`index.html`)
- Services (`services.html`)
- Contact (`contact.html`) with Formspree form

## How to Deploy on GitHub Pages
1. Create a new repository on GitHub.
2. Upload all files and folders from this project.
3. Go to **Settings > Pages**.
4. Select `main` branch and `/root` folder.
5. Save. Your site will be live at `https://username.github.io/repository-name`.

## Notes
- Replace `YOUR_FORMSPREE_ID` in `contact.html` with your actual Formspree form ID.
- Replace placeholder images in `images/` with your logo and hero image.
- Customize text and pricing in `services.html` as needed.
